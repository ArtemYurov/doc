# Breadcrumbs

The `moonshine::breadcrumbs` component is used to create `breadcrumbs` ("breadcrumbs").

```php
<x-moonshine::breadcrumbs
    :items="[
        '/' => 'Home',
        '/articles' => 'Articles'
    ]"
/>
```
