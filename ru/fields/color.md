## Основы

Наследует [Text](#/docs/{{version}}/fields/text.md).

\* имеет те же возможности

Поле Color является расширением Text, которое предоставляет удобный способ для ввода цвета.

```php
use MoonShine\UI\Fields\Color; 

Color::make('Color')
```
