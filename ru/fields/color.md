# Color

## Основы

Содержит все [Базовые методы](#/docs/{{version}}/fields/basic-methods.md).

Поле Color предоставляет удобный способ для ввода цвета.

```php
use MoonShine\UI\Fields\Color; 

Color::make('Color')
```
